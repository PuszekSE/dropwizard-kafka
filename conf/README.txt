--- DROPWIZARD RUN README ---

You can run this project in the following ways:
1. By 'mvn install', on 'stock-iface', 'dropwizard-kafka', then running it via 'mvn exec:java', and 'python stock-client.py <SYMBOLS>'.

2. Running 'start-dropwizard.sh' from scripts directory and after running 'start-reading-stockmarket.sh'.
